# Data Analysis & Visualization using Excel

## Project Overview
This project demonstrates data analysis and visualization using Microsoft Excel. A sales dataset was analyzed to identify trends across products, regions, and time.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Data Analysis Functions

## Key Features
- Data cleaning and structuring
- Pivot table analysis
- Interactive dashboard
- Business insights generation

## Insights
- Maggi is the top-performing product
- North region shows highest sales
- Sales trend increases over time
